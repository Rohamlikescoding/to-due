To-Due 🚀

A modern, responsive task management application designed to help users organize their daily workflow, track progress, and stay productive with a clean and intuitive interface.

To-Due focuses on providing a simple but powerful experience for managing tasks — from creating and completing tasks to tracking progress through an efficient state-managed architecture.

✨ Features

- ✅ Create, manage, and organize tasks
- ⏱️ Task timer functionality for focused productivity sessions
- ✔️ Mark tasks as completed
- 🗑️ Delete tasks when they are no longer needed
- 🔄 Persistent and predictable state management with Redux Toolkit
- ⚡ Fast and responsive React-based user interface
- 🎨 Clean UI with reusable components
- 🔐 User authentication state handling
- 📱 Responsive design for different screen sizes

🚀 Get Started
-Clone the repository
-Navigate into the project folder
-Install dependencies:
  npm i 
-Start the development server and fake api server:
  npm run dev
  npm run server
  
The application will be available at:
  http://localhost:5173
